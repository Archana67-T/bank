
# 🏦 BankGenius AI

**BankGenius AI** is an AI-powered Banking Exam Preparation Platform designed to help aspirants prepare for competitive banking examinations such as SBI PO, SBI Clerk, IBPS PO, IBPS Clerk, IBPS RRB, RBI Assistant, RBI Grade B, NABARD, and other banking exams.

The platform combines comprehensive study materials, daily banking current affairs, aptitude practice, reasoning, computer awareness, general knowledge, mock tests, and an AI-powered chatbot to provide a personalized and interactive learning experience.

# 🎯 Problem Statement

Banking aspirants often rely on multiple applications and websites to study banking awareness, current affairs, aptitude, reasoning, computer awareness, and general knowledge. Switching between platforms reduces productivity and makes learning inefficient.

BankGenius AI solves this problem by providing all essential banking exam preparation resources along with an AI-powered doubt-solving assistant in a single platform.
